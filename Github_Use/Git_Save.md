### Git版本管理

* git status
* 查看当前状态。如果你看到红色 X，说明这里有未提交的改动

* git add
* 将目录下所有的改动放人入暂存区

* git commit -m "shuoming" 
* 将改动打包发送存入本地数据库

* git push origin main 
* 将本地代码法法发送到github

* git log
* 查看历史提交记录

* git log --oneline
* 列出历史记录

* git checkout <Commit ID>
* 查看之前的存档

* git checkout main
* 回到现在的存档

* git reset --soft HEAD~1
* 撤销上一次提交

* git reset --hard HEAD~1
* 之后的存档其全部删除
